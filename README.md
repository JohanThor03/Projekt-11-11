## Information om HTML
HTML (HyperText Markup language), är ett språk som programmerare kan använda sig av för att designa och konstruera webbsidor. 
Med hjälp av HTML kan man skapa en layout av text och bilder i olika storlekar och typsnitt för sin hemsida. 
HyperText gör så att man kan länka sig mellan olika webbsidor på intranätet. 
HTML består av tre huvuddelar:

* html (Själva pappret som allt ligger på)
* Head (Högst upp på pappret finns ett huvud för rubriker och underrubriker)
* body (Där övriga saker som t.ex. brödtext, bilder och tabeller hamnar under)
